# GreenOS - **операционная система** для запуска на реальном ПК

## Что внутри
* ``` kernel.c ``` - Основное ядро системы. Запускается из ассемблерного загрузчика
* ``` boot.asm ``` - Ассемблерный загрузчик. Запускает основное ядро, Имеет встроенное сообщение об ошибке чтения диска
* ``` isr.s ``` - Обработчик прерываний. Не работает
* ``` \lib ``` - Папка с библиотеками.
* ``` linker.ld ``` - Конфигурация для линкира
* ``` run.sh ``` - Скрипт для запуска в Qemu
* ``` create.sh ``` - Скрипт для загрузки на флешку

## Как запустить:

Требуется:

### 1. Qemu

* Сам Qemu
* i686-elf **Компилятор**, **линковщик**, **objcopy** (Обычно скачиваются вместе)
* Nasm

#### Порядок действий:  

* Перед первым запуском надо создать пустой образ, делается командой: ``` dd if=/dev/null of=disc.img bs=1M count=1 status=progress ```, также вы можете зайти во вкладку релизов на GitHub и скачать образ оттуда
* Запустить скрипт **run.sh**

### 2. Реальный ПК

* i686-elf **Компилятор**, **линковщик**, **objcopy** (Обычно скачиваются вместе)
* Nasm
* Флешка
* Rufus

#### Внимание: скрипт для загрузки на флешку имеет **batch** формат и требует запуска на windows

#### Порядок действий: 

* Запустить скрипт **create.sh**
* Подключить к компьютеру флешку
* Дождаться появления окна от UAC (окно может появиться в свёрнутом виде)
* Разрешить Rufus изменения на этом устройстве
* Выбрать флешку 
* В папке с исходниками появится файл **os-image.bin**. Надо выбрать его в окне выбора файла
* Нажать кнопку **Старт** 
* Перезагрузиться
